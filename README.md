# Lintelligent Solution All Project API Documentation

This repository contains the API documentation for all Lintelligent Solution projects.

## Branch Structure

- **main**: Production-ready documentation
- **release**: Pre-production release candidate documentation
- **develop**: Development branch for ongoing documentation work

## Getting Started

This repository follows a standard Git flow branching model:
- All development work happens in the `develop` branch
- When ready for release, changes are merged to `release` branch
- After testing and approval, `release` is merged to `main` branch

## Contributing

Please create feature branches from `develop` and submit pull requests back to `develop`.
